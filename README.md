## fast-iseven-api

Fastest isEven API with Hono and Cloudflare Workers.(joke app)

```bash
$ curl  https://fast-iseven-api.mkan0141.workers.dev/api/iseven/1
{"iseven":false}

$ curl  https://fast-iseven-api.mkan0141.workers.dev/api/iseven/2
{"iseven":true}
```

## Usage

### development

```bash
$ yarn dev
```

### deploy

```bash
$ yarn deploy
```
